What is the Document Object Model(DOM)

The Document Object Model (DOM) is a structured representation of a web page, where each HTML element is treated as an object in a tree-like structure. JavaScript uses the DOM to interact with and manipulate web content dynamically, allowing developers to change elements, attributes, styles, and content on a page.

Why is event delegation in the context of the DOM and why it is useful

Event delegation is a technique in the DOM where instead of adding event listeners to individual child elements, you add a single listener to a common parent element
Reduces the number of event listeners, which is especially beneficial for dynamically created elements.

What is event bubbling in the context of the DOM?

Event bubbling is a type of event propagation in the DOM where an event starts from the most specific target element and "bubbles up" to the parent elements, eventually reaching the root element.
It allows you to handle events at a higher level, simplifying event handling for multiple child elements.
